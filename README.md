# PascalPointCloudViewer3d
Opens in Lazarus IDE Free Pascal 3d point clouds in Window.<br/>
It uses OpenGl.<br/>
(TOpenGLControl package must be installed).<br/>
Example point cloud is located in 'points.txt' file.<br/>
